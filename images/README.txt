PRODUCT PHOTOS GO HERE
======================

The website looks for a photo for each product in this folder. If a photo is
here, it shows up on the "would you still buy it?" screen — blurred (a censor /
mosaic effect) until the visitor clicks reveal, then it snaps to the clear photo.
If a photo is missing, the site falls back to an elegant text frame instead — so
nothing breaks.

HOW TO ADD A PHOTO
1. Find a product photo (e.g. Google Images, or the brand's own site).
2. Right-click it -> "Save Image As..."
3. Save it into THIS folder with the matching file name below.
4. Refresh the website. Done.

NOTE FOR THE PROJECT: downloading the images into this folder is the right way to
do it for the final version — they won't break and they work offline. Avoid
hot-linking images directly off Google search results; those URLs expire and many
sites block it. If you do use someone's photo, note the source in your write-up.
